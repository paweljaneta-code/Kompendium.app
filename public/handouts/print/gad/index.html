Standalone HTML do druku A4 — minimalistyczna instrukcja + ćwiczenie + miejsce na własne notatki.

Każdy handout to samodzielny plik HTML z osadzonymi fontami z CDN — można otwierać w przeglądarce i drukować bezpośrednio (Ctrl/Cmd + P). Dołączone wersje PDF jako fallback offline. Wzorzec wspólny: intro_box „Jak używać" / „Dlaczego to działa" u góry → skala PRZED → sekcje ćwiczenia → skala PO → callout. Pozostałe 7 narzędzi w module GAD to kwestionariusze (GAD-7, PSWQ, IUS-12), case'y, farmakoterapia, skala narzędzi, diff-diag — zgodnie z regułą wykluczone z kampanii handoutów.
